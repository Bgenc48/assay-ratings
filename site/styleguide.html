Design system

The tokens and components the Assay site is built from. Editorial and restrained: this is a ratings agency, not a trading app. Everything here is themed from CSS custom properties in css/site.css and adapts to light and dark automatically.

Reference page — not linked from the site nav and excluded from the sitemap. The grade palette is WCAG-AA for white text in both themes, and grades always carry their letter so colour is never the only signal.

Serif (system Iowan/Palatino/Georgia stack) for headings signals editorial authority; system sans for body; monospace for addresses, grades, and figures. No web fonts are fetched — CSP forbids external origins.
